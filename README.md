# Backend Evaluation

A backend system built with Node.js and TypeScript consisting of three components.

## Structure

- `logging_middleware/` - Reusable logging package that sends logs to the evaluation server
- `vehicle_maintence_scheduler/` - Optimises vehicle maintenance tasks using dynamic programming
- `notification_app_be/` - REST API for a campus notification system with priority inbox

## Tech Stack

- Node.js
- TypeScript
- Express.js
- Axios
